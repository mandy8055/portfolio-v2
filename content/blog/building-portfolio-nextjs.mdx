### Why This Structure?

1. **Separation of Concerns**: Content lives separately from code
2. **Type Safety**: Data files use TypeScript for IntelliSense
3. **Reusability**: UI components can be shared across sections
4. **Scalability**: Easy to add new features

## Key Technologies

### Next.js 15 App Router

The App Router brings several benefits:

- **Server Components by Default**: Better performance
- **Layouts**: Shared UI across routes
- **Streaming**: Progressive rendering
- **Built-in SEO**: Metadata API

### TypeScript

Type safety throughout:

```typescript

  slug: string;

  description: string;

  readingTime: string;
  published: boolean;

```

### Tailwind CSS 4

Utility-first CSS with:

- Dark mode support
- Responsive design
- Custom color schemes
- Optimized bundle size

## MDX Setup

MDX allows writing blog posts with JSX:

```mdx

```

### Blog Utilities

Created helper functions in `lib/blog.ts`:

- `getAllBlogPosts()`: Get all posts
- `getBlogPost(slug)`: Get single post
- `getAllTags()`: Get unique tags
- `searchBlogPosts(query)`: Search functionality

## Performance Optimizations

1. **Server Components**: Most components render on server
2. **Image Optimization**: Using `next/image`
3. **Font Optimization**: Using `next/font`
4. **Code Splitting**: Automatic with App Router
5. **Lazy Loading**: Dynamic imports for heavy components

## Dark Mode Implementation

Using `next-themes`:

```typescript
import { ThemeProvider } from 'next-themes';

export default function RootLayout({ children }) {
  return (
    <html suppressHydrationWarning>
      <body>
        <ThemeProvider attribute='class'>{children}</ThemeProvider>
      </body>
    </html>
  );
}
```
